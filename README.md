# Eco Negocios e Inversiones

Plataforma de inversiones sostenibles en proyectos de energía renovable.

## Dashboard

- Vista Inversor: Resumen, Depósito, Retiro
- Vista Admin: Transacciones, Gráficos, Beneficiarios

## 🚀 Commands

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `npm install`     | Installs dependencies                       |
| `npm run dev`     | Starts local dev server at `localhost:4321` |
| `npm run build`   | Build production site to `./dist/`          |
